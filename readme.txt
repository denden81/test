ТЗ
Задание:
Используя средство автоматизации тестирования web-приложения (selenium + python),
напишите тест, состоящий из следующих шагов:

- Открыть страницу регистрации хостинг-аккаунта на сайте http://timeweb.com/.

- Зарегистрировать учетную запись.

- Проверить правильность регистрации на выбранный тарифный план и соответствие его характеристик отображаемым на сайте.

Составьте инструкцию по запуску получившегося теста.

Информация по тесту

Тест написан на Python
При написании использовался:
python-3.6.3
pycharm-community-2017.2.3

для запуска теста необходимо установить python-3.6.3,pycharm-community-2017.2.3
а так же скопировать в папку с установленным python-3.6.3 файлы chromedriver и geckodriver

при запуске теста необходимо выбрать нужный тариф и период (по умолчанию стоит optimo и оплата за год)
нужный тариф расскоментируем, дефолтные коментируем.

Кейс теста:

1. Выбирается тариф и период
2. Генерируется рандомное число для емайл адреса
3. Переключается тариф на выбранный период
4. Считываем данные выбранного тарифа со страницы
5. Вводится ФИО и почта, происходит регистрирование и вход в личный кабинет
6. Проверяем начальное поздравление о регистрации и закрываем его 
7. Считываем данные выбранного тарифа с личного кабинета
8. Сравниваем данные со страницы с данными из личного кабинета
9. Если данные совпадают то выводим сообщение об совпадении данных, в противном случае выводим сообщение о несовпадении данных
Бонус
10. Переходим в почтовый ящик и чекаем письмо о регистрации
11. Удаляем все письма из ящика.
